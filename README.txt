各ファイルの説明

・RTC
　あっち向いてほいシステムで利用するRTC群

    ・Finger_Point
 　  あっち向いてほいの指の動作を行うコンポーネント

    ・Game_Judge
 　  あっち向いてほいの勝敗判定を行うコンポーネント

    ・KinectFace
　   Kinectの機能を用いて，顔方向をroll,pitch,yawで出力するコンポーネント

    ・Direction
	顔方向(roll,pitch,yaw)から顔方向を決定するコンポーネント

    ・communication
     ルールの説明，開始の合図，勝敗の伝達を行うコンポ―ネント


・Document
　 各種マニュアルが入ったファイル
